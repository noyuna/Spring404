# HereJi Self-Demo Guide

## 1. 데모 목적

본 문서는 배포된 HereJi 서비스를 교수님께서 직접 확인하실 수 있도록 작성한 self-demo 안내서입니다.
HereJi는 사용자 리뷰와 공공 안전 데이터를 기반으로 장소별 안전 점수를 제공하고, 출발지와 목적지 사이의 보행 경로 중 더 안전한 경로를 추천하는 데이터 기반 Safety Map 서비스입니다.

## 2. 배포 링크

- Frontend Demo: `<프론트엔드 배포 링크를 입력하세요>`
- Backend API Docs: `<백엔드 Swagger 문서 링크를 입력하세요>`
- AI API Docs: `<AI 서버 Swagger 문서 링크를 입력하세요>`

> 교수님께서 직접 시연하실 때는 Frontend Demo 링크만 접속하시면 됩니다.
> Backend API Docs와 AI API Docs는 기술 구조 확인용 링크입니다.

## 3. 권장 시연 환경

- Chrome 브라우저 사용 권장
- 위치 권한 요청 시 허용 권장
- 데모 지역은 홍대 인근을 기준으로 구성되어 있습니다.
- 배포 서버가 Render 무료 인스턴스인 경우, 첫 접속 시 서버가 깨어나는 데 30초~1분 정도 소요될 수 있습니다.

## 4. Self-Demo 시연 순서

### Step 1. 서비스 접속

1. Frontend Demo 링크에 접속합니다.
2. 지도 화면이 로딩되는지 확인합니다.
3. 화면 하단의 HereJi Safety Map 안내 영역을 확인합니다.

확인 포인트:

- Google Map이 정상적으로 표시되는지
- 현재 위치 버튼과 하단 정보 패널이 보이는지
- 장소 검색 또는 지도 클릭이 가능한지

### Step 2. 장소 선택 및 안전 점수 확인

1. 지도에서 홍대 인근 장소를 클릭하거나 검색창에 장소명을 입력합니다.
2. 장소를 선택하면 하단 패널에 장소명, 주소, 평균 평점, 안전 점수가 표시됩니다.
3. 기존 리뷰가 있는 경우 리뷰 목록을 확인합니다.

확인 포인트:

- 장소별 안전 점수가 표시되는지
- 사용자 리뷰가 장소 상세 화면에 표시되는지
- 안전 점수가 5점 만점 기준으로 표시되는지

### Step 3. 사용자 리뷰 작성

1. 장소 상세 화면 하단에서 별점을 선택합니다.
2. 리뷰 내용을 입력합니다.
   - 예시: `골목이 조금 어둡고 사람이 적어서 불안했습니다.`
   - 예시: `주변에 편의점과 가로등이 많아서 비교적 안전하게 느껴졌습니다.`
3. `리뷰 저장` 버튼을 클릭합니다.
4. 저장 후 리뷰 목록과 안전 점수가 갱신되는지 확인합니다.

확인 포인트:

- 사용자가 입력한 리뷰가 저장되는지
- AI 서버가 리뷰 문장을 분석하여 ai_score를 생성하는지
- 사용자 별점, AI 분석 점수, 공공 안전 데이터가 결합되어 최종 안전 점수가 갱신되는지

### Step 4. 출발지와 도착지 설정

1. 장소 상세 화면에서 `출발` 버튼을 눌러 출발지를 설정합니다.
2. 다른 장소를 검색하거나 지도에서 선택합니다.
3. 선택한 장소에서 `도착` 버튼을 눌러 도착지를 설정합니다.
4. 출발지와 도착지가 하단 패널에 표시되는지 확인합니다.

확인 포인트:

- 출발지와 도착지가 정상적으로 설정되는지
- 지도에 출발/도착 마커가 표시되는지
- 현재 위치를 출발지로 사용할 수 있는지

### Step 5. 안전 경로 추천 확인

1. 출발지와 도착지를 설정하면 보행 경로 후보가 생성됩니다.
2. 각 경로의 거리, 예상 시간, 안전 점수, 주변 리뷰 수를 확인합니다.
3. 가장 안전 점수가 높은 경로가 상단에 추천되는지 확인합니다.
4. 경로를 선택하면 지도 위의 경로 표시가 변경되는지 확인합니다.

확인 포인트:

- 여러 경로 후보가 표시되는지
- 경로별 안전 점수가 표시되는지
- 최적 추천 경로가 안전 점수 기준으로 정렬되는지
- 지도 위에 선택한 경로가 시각화되는지

## 5. 핵심 구현 흐름

```text
사용자
 ↓
React 지도 화면
 ↓
FastAPI 백엔드
 ├─ MySQL: 리뷰 및 공공 안전 데이터 저장/조회
 └─ AI 서버: 리뷰 문장 안전도 분석
 ↓
구역별 최종 안전 점수 계산
 ↓
안전 경로 추천 및 지도 시각화
```

## 6. 기술 구현 요약

- Frontend: React 기반 지도 UI, 장소 검색, 리뷰 작성, 출발지/도착지 설정, 경로 시각화
- Backend: FastAPI 기반 API 서버, 리뷰 저장, 위치 좌표 기반 zone 계산, 최종 안전 점수 산출
- Database: MySQL 기반 리뷰 데이터, 공공 안전 데이터, 지도 구역 데이터 저장
- AI Server: OpenAI API 기반 리뷰 문장 분석 및 ai_score 산출
- Route Logic: 경로가 지나가는 구역의 안전 점수를 평균화하여 안전 경로 추천

## 7. 데모 중 확인할 핵심 가치

- 사용자는 지도에서 특정 장소의 안전도를 직관적으로 확인할 수 있습니다.
- 사용자가 작성한 리뷰가 AI 분석을 거쳐 안전 점수에 반영됩니다.
- CCTV, 보안등, 편의점 등 공공 안전 데이터가 장소별 안전 점수 계산에 활용됩니다.
- 단순 최단 경로가 아니라 안전 점수를 고려한 보행 경로를 선택할 수 있습니다.

## 8. 문제 발생 시 참고 사항

- 첫 접속 시 화면 로딩이 느린 경우 Render 서버가 대기 상태에서 깨어나는 중일 수 있습니다.
- 안전 점수나 리뷰가 바로 표시되지 않는 경우 새로고침 후 다시 시도해 주세요.
- 위치 권한을 거부한 경우 현재 위치 기반 출발지 설정이 제한될 수 있습니다.
- 경로 추천이 표시되지 않는 경우 출발지와 도착지를 모두 홍대 인근 장소로 설정해 주세요.

## 9. 한 줄 요약

HereJi는 사용자 리뷰, AI 분석, 공공 안전 데이터를 결합해 장소별 안전 점수를 제공하고, 이를 기반으로 더 안전한 보행 경로를 추천하는 데이터 기반 안전 지도 서비스입니다.
